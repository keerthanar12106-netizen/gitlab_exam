# gitlab_exam
this repo is for external  exam purpose only
I have used the commands:
"git global config"-to config my repo to the git bash.
repo link can also be added usung this command"git remote add origin"
"git init"-Create an empty Git repository or reinitialize an existing one.
"git clone"-it is used to clone my repo to the local sysytem. 
"git checkout -b<branch-name>"- it creates the new branch.
"git commit -message"-it records the changes made to the repository.
"git push origin main "- it pushes the file to the repo link added. 
"git pull origin main"- it   Fetch from and integrate with another repository or a local branch.
"git fetch origin "- it fetches all the commit made recently ot he repository. 
"git rebase origin"-shows recent commit
"git log"-it shows all the commits made to the repo.
"git show SHA"- shows the perticular commit made.
